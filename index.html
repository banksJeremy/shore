Shore Math Library and [Noun]

Shore is a math library being implemented in CoffeeScript. A browser interface is included.

The interface is being developed in Chrome, but is intended to be functional in all relatively modern browsers when it's finished.

Documentation and this page are a low priority since the basic functionality isn't finished yet. It'll be nicer later. For now more details might be available in the README file, which you can view on GitHub.

Shore is developed by Jeremy Banks, the source and full credits are on GitHub.
